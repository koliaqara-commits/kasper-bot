# Каспер Telegram bot

Бот для сделок с NFT-подарками: создание сделки, ссылка для покупателя, балансы пользователей и админское пополнение по ID.

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Создать `.env` по примеру `.env.example`.

3. Запустить:

```bash
python bot.py
```

## Админ-команды

```text
/admin
/addbalance <telegram_id> <amount> [Stars|TON|USDT|RUB]
/balance <telegram_id>
/setpaylink <url>
```

Кнопка оплаты появляется только у покупателя сделки. Если чужой пользователь откроет ссылку сделки, он увидит сообщение, что сделка предназначена другому покупателю.
